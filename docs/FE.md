## UI Library

The application leverages [Untitled UI](https://untitledui.com/) as its primary UI library. Untitled UI offers a comprehensive set of pre-designed components that help build a consistent and visually appealing user interface.

Unlike other component libraries, Untitled UI is not a pre-built library. Instead, it provides a collection of design tokens, utility classes, and base components that are installed into the project.

The `src/governance-app-frontend/untitledui` directory contains the raw code generated by Untitled UI. Components are installed in a designated folder, while utilities and styles are added to the `/src` directory. We manually move them into the `untitledui` folder to keep them organized and separate from the rest of the application code.

Additionally, we manually manage an `index.ts` file to export all base components without the need for deep imports. This file is not generated by Untitled UI and must be updated whenever new components are added or removed.
